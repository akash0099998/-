🌟 About Goat Bot V2

Goat Bot V2 is a powerful Facebook Messenger automation bot designed for:

- Group management
- Entertainment commands
- Automation tools

This repository is a custom modified version maintained by Mohammad Akash.

---

⚙️ Framework Information

- Framework: Goat Bot V2
- Platform: Facebook Messenger
- Language: Node.js
- Edition: Custom Modified

---

🛠 Features

- 🤖 Auto Reply System
- 🎭 Welcome & Leave Messages
- 📦 Custom Commands
- 🛡️ Admin Tools
- 🌍 Multi Language Support
